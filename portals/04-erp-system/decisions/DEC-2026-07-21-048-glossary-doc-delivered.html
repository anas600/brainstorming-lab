🎨 المنهجية (Methodology)

7 قواعد رسمية للترجمة:

  1. الأولوية للعربية الفصحى المبسّطة — لا عامية صعبة، لا فصحى معقّدة
  2. مصطلحات IFRS معتمدة كما هي (مثل "ميزان المراجعة")
  3. مصطلحات تقنية = EN بين قوسين في أول ذكر
  4. أرقام الحسابات تبقى أرقام (لا تُترجم 1100، 1300...)
  5. API endpoints تبقى EN (لا تُترجم /api/finance/...)
  6. Event types تبقى EN (StockReceived، PaymentReceived)
  7. للمتناقضات: ملاحظة ⚠️ صريحة + اقتراح حل

متى نستخدم "مفضّل" vs "بديل"؟

المفضّل البديل السبب
ترحيل Posting العربي أوضح
حذف ناعم Soft Delete الأكثر استخداماً عربياً
حقوق ملكية Rapatrié الأول فصحى
متعهد به Committed الأول مترجم
