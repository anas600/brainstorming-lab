1) محاسبة (Accounting)

📐 منهجية الترجمة (Translation Methodology)

القواعد المعتمدة:

  1. الأولوية للعربية الفصحى المبسّطة — لا عامية صعبة، لا فصحى معقّدة
  2. مصطلحات محاسبية معتمدة من IFRS — مثل "ميزان المراجعة"، "حقوق الملكية"
  3. مصطلحات تقنية = EN بين قوسين في أول ذكر (مثل: "نمط Outbox (Outbox Pattern)")
  4. أسماء الحسابات (1100، 1300...) تبقى كأرقام — لا تُترجم
  5. API endpoints تبقى EN — /api/finance/posting-rules لا تُترجم
  6. Event types تبقى EN — StockReceived, PaymentReceived — لأن الكود يستخدمها
  7. للمتناقضات: نضع تنبيه ⚠️ ونقترح الحل (مثل SAR/LYD)

متى نستخدم "مفضّل" vs "بديل"؟

المفضّل البديل السبب
ترحيل Posting العربي أوضح للقارئ غير التقني
حذف ناعم Soft Delete الأكثر استخداماً في التوثيق العربي
حقوق ملكية Rapatrié الأول فصحى، الثاني فرنسي (نادر)
متعهد به Committed الأول مترجم، الثاني شائع في الندوات
